Task Manager Web App

A simple and clean Task Manager web application built with Django.
Users can create, update, delete, search, and manage daily tasks easily.

---

Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Task status management
  - Pending
  - In Progress
  - Completed
- Due date support
- Search tasks
- Filter tasks by status
- Clean dashboard UI
- Homepage with features section

Installation

1. Clone the repository

git clone https://github.com/vinodsuthar01/task-manager.git](https://github.com/vinodsuthar01/task_manager

2. Move into project folder

cd task-manager

3. Install dependencies

pip install -r requirements.txt

4. Run migrations

python manage.py migrate

5. Start server

python manage.py runserver

6. Open browser

http://127.0.0.1:8000/
